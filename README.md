# sogserver
The server part which should be used to create a persistent environment for oglike home edition.

# Installation

- Clone the repo: `git clone git@github.com:Knoblauchpilze/sogserver.git`
- Go to the project's directory `cd ~/path/to/the/repo`
- Compile: `make r`
- Install: `make install`

# Usage

Don't forget to add `/usr/local/lib` to your `LD_LIBRARY_PATH` to be able to load shared libraries at runtime.
